檔案進行打包:

  • 其他常用指令
    • -F → 打包為單一可執行文件
    • -i → 圖示，只適用於 Windows 平台
    • -w →不出現命令行
    • -D → 打包為一個文件夾
    • -n → 生成文件的名稱
      pyinstaller 執行後會生成兩個資料夾 build 和 dist 跟一個 spec 文件
      其中 build 裡面的 warn-xxx.txt 可查看執行 pyinstaller 過程的錯誤訊息，
      xref-xxx.html 可查看 import 模塊的依賴關係。

生成的 EXE 檔案會存放在 dist 資料夾裡。

  • 加入其他附加檔案

  • 不顯示命令列 → console = False
